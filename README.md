# EchoMusicWeb

- A web project work with echo-music-api
- Base on react, tailwindcss and vite

## Started

```bash
pmpm install
```

---

## Develop

```bash
pnpm dev
```

---

## Build

```bash
pnpm build
```

---

## Config

- Config api baseUrl

replace `VITE_API_URL` value in `.env.local` file

- Config web owner email

replace `VITE_EMIAL` value in `.env.local` file
